apps.0install.net - Development tools

To add git to your PATH:

      $ 0alias git https://apps.0install.net/devel/git.xml

To integrate Git and related tools in your Windows desktop environment:

      $ 0install integrate https://apps.0install.net/devel/git-for-windows.xml

To add common Google Cloud command-line tools to your PATH:

      $ 0alias gcloud https://apps.0install.net/devel/gcloud
      $ 0alias gsutil https://apps.0install.net/devel/gcloud gsutil
      $ 0alias bq https://apps.0install.net/devel/gcloud bq
