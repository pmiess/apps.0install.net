apps.0install.net - Python

Python is an interpreted, interactive, object-oriented programming language.

To add python to your PATH:

      $ 0alias python https://apps.0install.net/python/python

To specify Python as the runner in a feed:

      <command name="run" path="myapp.py">
        <runner interface="https://apps.0install.net/python/python"/>
      </command>
