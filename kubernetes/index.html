apps.0install.net - Kubernetes tools

Kubernetes is an open source system for managing containerized applications across multiple hosts.

To add the Kubernetes client CLI to your PATH:

      $ 0alias kubectl https://apps.0install.net/kubernetes/kubectl

To add additional common Kubernetes tools to your PATH:

      $ 0alias minikube https://apps.0install.net/kubernetes/minikube
      $ 0alias helm https://apps.0install.net/kubernetes/helm

To make 0install automatically download a Helm (client) version that matches the Tiller (server) version:

      $ 0install add helm https://apps.0install.net/kubernetes/helm-autoversion

To have 0install automatically pick Helm versions when pulled in as a dependency (e.g., from helmfile):

      $ 0install add-feed https://apps.0install.net/kubernetes/helm https://apps.0install.net/kubernetes/helm-autoversion
