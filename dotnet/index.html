apps.0install.net - .NET

Command-line tools

Available feeds

  • DocFX (documentation generation tool)
  • NuGet (package manager for .NET)
  • NAnt (free .NET build tool)

Samples

To add nuget (.NET Package Manager) to your PATH:

      $ 0alias nuget https://apps.0install.net/dotnet/nuget

To add nant (free .NET build tool) to your PATH:

      $ 0alias nant https://apps.0install.net/dotnet/nant

.NET Core

Available feeds

  • .NET Core Runtime
  • .NET Core SDK

Samples

To use .NET Core as the runner in a feed:

      <command name="run" path="myapp.dll">
        <runner interface="https://apps.0install.net/dotnet/core"/>
      </command>

To add dotnet (.NET Core SDK) to your PATH:

      $ 0alias dotnet https://apps.0install.net/dotnet/core-sdk

Classic .NET

Available feeds

  • Common Language Runtime
  • Common Language Runtime with MONO_PATH support
  • Mono
  • .NET Framework
  • .NET Framework Client Profile
  • IronPython

Samples

To use a Common Lanaguage Runtime (Microsoft's .NET Framework or Mono) as the runner in a feed:

      <command name="run" path="myapp.exe">
        <runner interface="https://apps.0install.net/dotnet/clr"/>
      </command>

To use either Microsoft's .NET Framework with emulated MONO_PATH support or Mono as the runner in a feed:

      <command name="run" path="myapp.exe">
        <runner interface="https://apps.0install.net/dotnet/clr-monopath"/>
      </command>
      <requires interface="http://mylib/">
        <environment name="MONO_PATH" insert="."/>
      </requires>

To use the full version of Microsoft's .NET Framework as the runner in a feed:

      <command name="run" path="myapp.exe">
        <runner interface="https://apps.0install.net/dotnet/framework"/>
      </command>

To use either the Client Profile or the full version of Microsoft's .NET Framework as the runner in a feed:

      <command name="run" path="myapp.exe">
        <runner interface="https://apps.0install.net/dotnet/framework-client-profile"/>
      </command>

To use Mono as the runner in a feed:

      <command name="run" path="myapp.exe">
        <runner interface="https://apps.0install.net/dotnet/mono"/>
      </command>

To use IronPython (Python implementation running on the CLR) as the runner in a feed:

      <command name="run" path="myapp.py">
        <runner interface="https://apps.0install.net/dotnet/ironpython"/>
      </command>
